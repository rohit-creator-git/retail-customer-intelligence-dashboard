# 📊 Retail Customer Intelligence Dashboard

## Overview

This project analyzes retail sales data using SQL, Python, Microsoft Excel, and Power BI. The objective is to identify sales trends, customer behavior, regional performance, and profitability through data analysis and interactive dashboards.

## Tools & Technologies

- SQL (MySQL)
- Python
- Microsoft Excel
- Power BI
- Pandas
- NumPy
- Matplotlib

## Dataset

Sample Superstore Dataset

- Total Records: 9,994
- Total Columns: 13

## Project Objectives

- Analyze Total Sales
- Analyze Total Profit
- Category-wise Sales Analysis
- Regional Performance Analysis
- Customer Segment Analysis
- Interactive Dashboard Development

## Business Insights

- Consumer segment generated the highest sales.
- Technology category showed strong revenue performance.
- Discounts negatively impacted profitability in several categories.
- Regional sales performance varied across different locations.
